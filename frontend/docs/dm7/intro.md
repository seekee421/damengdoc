## 🔧 系统要求

### 硬件配置
- **CPU**: x86_64 架构，1核心以上
- **内存**: 2GB 以上（推荐 4GB）
- **存储**: 10GB 以上可用空间
- **网络**: 百兆以太网以上

### 操作系统支持
- **Linux**: 
  - Red Hat Enterprise Linux 6.x/7.x
  - CentOS 6.x/7.x
  - SUSE Linux Enterprise Server 11/12
  - Ubuntu 14.04/16.04/18.04
- **Windows**: 
  - Windows Server 2008 R2/2012/2016
  - Windows 7/8/10
- **Unix**: 
  - AIX 6.1/7.1
  - Solaris 10/11
